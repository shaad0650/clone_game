# Mirror Clone Survival

**"Your past actions become future enemies."**

A strategic top-down survival shooter where every move you make is recorded and replayed by enemy clones. Survive against your own timeline, use strategic Time Echoes to create allies, and battle massive Boss Clones.

## üéÆ Features

*   **Time Loop Mechanics**: Enemies are clones of *you* from the past. They move, shoot, and dash exactly as you did.
*   **Time Echo Ability**: Press **E** to spawn a friendly clone from your past timeline to fight alongside you.
*   **Clone Mutations**: Enemies can mutate into **Fast**, **Mirror**, or **Unstable** variants.
*   **Boss Battles**: Every 30 seconds, face a massive **Boss Clone** with extra health and speed.
*   **Progression System**: Earn XP to level up, unlock new weapons (Spread Gun, Laser), and choose upgrades.
*   **Environmental Hazards**: Survive against Moving Walls, Rotating Lasers, and Danger Zones.
*   **Replay System**: Watch a full replay of your run after Game Over.

| Key | Action |

| **WASD / Arrows** | Move |

| **SPACE** | Shoot in facing direction / Start Game |
| **SHIFT** | Dash (Invulnerable) |
| **E** | **Time Echo** (Spawn friendly clone) |
| **1 / 2 / 3** | Select Upgrade (when prompted) |
| **P** | Watch Replay (Game Over screen) |
| **R** | Restart Game |
| **ESC** | Quit |
